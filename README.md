# Web app to add books you have read

## Functionality
* create new user
* login
* create new book:
    - isbn number,
    - title,
    - publication year,
    - author,
    - publisher,
    - first page of book (image)
* edit existing book
* delete existing book
* view collections of your books
* validation : 
    - fields not blank,
    - isbn number have 13 numbers from 0 to 9,
    - user or book exist in the database,
    - password matching
